    Q&A

    Name 3 diffreneces between web 1.0, web 2.0 and web 3.0:

    1. Level of interaction: Web 1.0 is considered "read only web" , meaning there isn't real interaction between readers or publishers or each other. Web 2.0, the "read-write" web, enables interaction with other web users through social networking, blogging, wikis, tagging, user generated content, video etc. In web 3.0, using the user's participation, collaboration and information sharing, alongside evolving AI methods, the web can learn the user preferences and deliver personalized web experience.
    2. Static vs Dynamic: In web 1.0 we can encounter mainly static pages and sites, which is updated once in a while. It enables website owners to make their information available to anyone at anytime. In web 2 and web 3 , on the other hand, the sites are much more dynamic and often updated.
    3. Information search abilities: In web 1.0 a search of information will retrieve macro content which is not definitely accurate. In web 2.0, for example, we can search Google which will retrieve micro content with pretty good results. However, it will not necessarily be what the user searched for. In web 3.0 thanks to use of Semantic search, Machine learning& NLP, the retrieved information will probably be more satisfiable and relevant for the user.

    Example of a use-case: "Ask the web"- "Dance lessons":

    Web1- Will return a variety of static pages offering and discussing dance lessons, when some of them are no longer relevant. For those which the user will find relevant, he will not be able to filter by price, location, etc.
    Web2- More accurate pages will be retrieved, including Facebook pages of relevant dance teachers or dance groups, recommendations of users and designated web-sites which summarize all dance lessons options. As a user, I will be given the ability to filter the given lesson offer to my needs or interact with other users to get feedback, help and recommendations.
    Web3- In web3, the retrieved results will be more personal. The top results will be of dance lessons which are taking places near my home address or dance lessons of Hip-Hop and Salsa, which I've searched in the past. I will also be suggested to buy new dance shoes in the shop next to my house or to watch the new dance videos of my favorite studio!
